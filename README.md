## Playable Windows Build

`Cyber_Chess_0.5/` is the ready-to-run Electron game folder.

Run:

```text
Cyber_Chess_0.5/Codex_Cyber_Chess_0.5.exe
```

This is a playable `.exe` desktop build of **Cyber Chess 0.5** with the full soundtrack bundled into the app resources.

Game and soundtrack by **FrostD4D** and **Codex 5.5**.
